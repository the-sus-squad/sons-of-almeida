## Naming Conventions

### Git Branch Conventions

- **Feature Branch:** feature/new-feature-branch-name 
- **Bug Fix Branch:** bug/bug-fix-branch-name

### Code Style Conventions

- **Class Name:** PascalCase
- **Function Name:** PascalCase
- **Variable Name:** camelCase
